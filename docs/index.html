Burgtheater · Live Performance

KI-Debatte trifft Bühnenregie

Zwei KIs diskutieren live — während ein Director-System Video, Sound und Licht in Echtzeit auslöst. Operator im Loop: Autopilot, Overrides, Emergency Stop.

Repository auf GitHub Schnellstart lesen

Was die Maschine macht

Text wird zur Dramaturgie — Dramaturgie wird zu Cues — Cues werden zu OSC, MIDI und Licht am Pult.

Textquelle

KI-Debatte

GPT und Claude diskutieren abwechselnd über ein Thema. Optional mit TTS — Siri auf dem Mac oder edge-tts in Docker.

Regie

Director Pipeline

Regeln und LLM wählen Clips, Sounds und Lichtstimmungen. Der Scheduler achtet auf Abstände, Safety und Operator-Overrides.

Ausgabe

Live auf der Bühne

Pixera für Video, MIDI/Ableton für Ton, EOS per TCP für Licht. TouchDesigner optional im Signalweg.

Zwei Inszenierungs-Modi

Teil 1 für den Stücktext-Workshop — Teil 2 für die AVATAR-Aufführung mit Text-Sync.

Teil 1 — Stücktext

  • Dramaturgen-Workshop → Stücktext → Aufführung
  • Sequentiell: Diskussion, dann Stücktext
  • Pixera-Clips per Dramaturgie-Entscheidung
  • UI: /dramaturgie, /auffuehrung

Teil 2 — Text-Sync

  • AVATAR-Text hochladen → Vorbereiten → Aufführung
  • Eine KI-Stimme als Master-Clock
  • Avatar-Video feuert am CSV-Stichwort im Text
  • UI: /inszenierung, /inszenierung/auffuehrung
